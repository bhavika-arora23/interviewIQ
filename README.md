# 🚀 InterviewIQ

InterviewIQ is a full-stack AI-powered platform designed to help users prepare for interviews. It provides mock interview experiences, question generation, and evaluation features to improve technical and communication skills.

---

## ✨ Features

- 🎤 AI-powered mock interviews
- 🤖 Dynamic interview question generation
- 📊 Performance evaluation & feedback
- 👤 User authentication system
- 💳 Razorpay payment integration (if applicable)
- 📁 Dashboard for managing interviews
- 🎯 Role-based interview practice (optional)

# How to use InterviewIQ

### 1. Sign Up / Login
- Open the application
- Create a new account or login using existing credentials

### 2. Choose Interview Type
- Select the role you want to practice for (e.g., Frontend, Backend, Full Stack, etc.)

### 3. Start Mock Interview
- Click on "Start Interview"
- Answer the AI-generated questions in real time

### 4. Get Feedback
- After completing the interview, you will receive:
  - Performance score
  - Feedback on answers
  - Suggestions for improvement

### 5. Track Progress
- Go to your dashboard to see past interviews and improvement history
